AI辅助开发日志

在本次《文档自动排版工具》课程设计中，我使用了AI工具（ChatGPT）辅助完成部分代码设计与功能实现。

在开发初期，通过AI帮助我将系统拆分为文件读取、文档排版、字数统计和文件保存等模块，并确定采用“主控制器 + 功能模块”的结构来组织代码。

在具体实现过程中，AI帮助我生成了字符串处理相关代码，例如去除多余空格、删除空行以及基础文本格式整理等功能，同时也对字数统计功能的实现思路进行了指导。

在调试过程中，我发现各模块之间数据传递不够清晰，后来在AI建议下使用统一的content变量作为数据载体，使各功能模块能够围绕同一文本进行处理。

此外，本项目使用GitHub进行版本管理，完成了仓库创建、代码提交、分支管理以及版本同步操作，加深了对Git基本工作流程的理解。

通过本次开发，我对Java模块化设计和Git版本控制有了更直观的认识，也提高了独立完成小型项目的能力。
